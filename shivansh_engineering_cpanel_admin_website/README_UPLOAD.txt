SHIVANSH ENGINEERING - PHP ADMIN PANEL WEBSITE

Upload steps in cPanel:
1. Open cPanel > File Manager > public_html.
2. Upload this ZIP file.
3. Extract the ZIP inside public_html.
4. Ensure files like index.php, about.php and admin folder are directly inside public_html.
5. Open your website.
6. Admin panel: https://yourdomain.com/admin/login.php

Default admin login:
Email: admin@shivanshengg.com
Password: ChangeMe@123

IMPORTANT:
Change the admin password immediately from Admin Panel > Change Password.
Do not share passwords in chat or email.

To update website:
- Company/contact/home content: Admin > Company & Contact
- Products: Admin > Products
- Projects: Admin > Projects
- Gallery photos: Admin > Gallery

This version uses PHP + JSON files and works on most cPanel hosting with PHP enabled.
